# My First GitHub Project

This is my first GitHub project created while learning Git and GitHub.

## About

A simple HTML webpage that displays a welcome message and demonstrates the basic structure of an HTML document.

## Features

* Basic HTML5 structure
* Heading elements (`h1`, `h2`)
* Paragraph content
* Beginner-friendly project

## Technologies Used

* HTML5
* Git
* GitHub

## Project Structure

```text
.
├── app.html
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open `app.html` in any web browser.

## Learning Goals

* Understanding Git basics
* Creating a Git repository
* Making commits
* Pushing code to GitHub
* Managing project files
